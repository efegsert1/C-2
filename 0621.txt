// : 주석
코드에 설명을 달기 위한 문장이다.
컴퓨터는 주석을 무시하고 실행하지 않는다.

왜쓰냐?
코드 설명 : 코드가 무슨 듯인지 적어놓기
협업 : 다른 사람이 코드를 이해할 수 있도록
기록 : 나중에 다시 볼 때 쉽게 이해할 수 있도록
테스트 : 특정 코드를 일시적으로 비활성화 할 때 사용한다.
			(잠깐 사용하지 않을때)

주석의 종류

// : 한줄 주석

여러줄 주석
/*  */

자료형 종류 5가지
    자료형 이름 + 값(자료형에 맞는 값 아무거나
1. int a = 	1;
2. float 		3.14f;
3. string 		"hi";
4. char 		'A';
5. bool 		false;

1. 오늘 배운 내용 정리
1.출력문
정의 : 
콘솔에 내용을 출력하고 줄바꿈O : Console.WriteLine();
콘솔에 내용을 출력하고 줄바꿈X : Console.Write();

2.주석
정의: 
한줄 주석: //
여러줄 주석: /* */

3.입력문
= Console.ReadLine(); 문자
= int.Parse(Console.ReadLine()); 숫자

4.변수의 정의
변수는 값을 담는 상자 같은거

5.변수 문법
자료형 변수명 = 변수값;

6.자료형 타입 5가지
int, float, string, bool, char

7.산술연산자
+,-,*,/,%

8.비교연산자
>,<,>=,<=,==
